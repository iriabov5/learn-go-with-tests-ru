# Learn Go with Tests (на русском)

## Что это за книга

**"Learn Go with Tests"** — это бесплатная open-source онлайн-книга, написанная Крисом Джеймсом (Chris James). Книга учит языку программирования Go через методологию TDD (Test-Driven Development). Уникальный подход заключается в том, что каждая новая концепция языка вводится через написание тестов — сначала тест, потом реализация.

### Первоисточник

- **Официальный сайт:** https://quii.gitbook.io/learn-go-with-tests
- **GitHub репозиторий:** https://github.com/quii/learn-go-with-tests
- **Лицензия:** CC BY-NC-SA 4.0 (Creative Commons Attribution-NonCommercial-ShareAlike)

Книга охватывает путь от "Hello, World" до полноценных веб-приложений с базой данных, уделяя особое внимание:
- Стандартной библиотеке Go
- Архитектурным паттернам
- Написанию чистого и тестируемого кода
- Практическим примерам (CLI-утилиты, HTTP-серверы)

## Цель проекта

Мы создаём русскоязычную адаптацию книги с двумя ключевыми особенностями:

### 1. Перевод на русский язык
- Адаптация технических терминов
- Сохранение оригинальной структуры и последовательности глав
- Сохранение педагогического подхода (через тесты)

### 2. Сравнение с Java
Каждая глава будет содержать раздел "Аналог в Java", который показывает:
- Как та же концепция реализуется в Java
- Плюсы и минусы каждого подхода
- Идиоматичные решения на обоих языках
- Примеры кода на Java (с JUnit 5)

Это поможет Java-разработчикам быстрее освоить Go, используя существующие знания, и понять различия в философии языков.

## Структура книги (план)

### Введение
- Основы Go
- Почему TDD
- Настройка окружения

### Главы

#### 1. Hello, World
- **Go:** Первая программа, тестирование, запуск
- **Java:** Сравнение с `System.out.println`, JUnit basics

#### 2. Целые числа и циклы
- **Go:** `for`, `int`, диапазоны
- **Java:** `for`, `int`, Stream API

#### 3. Структуры и методы
- **Go:** `struct`, методы, приёмники
- **Java:** классы, методы, this

#### 4. Интерфейсы
- **Go:** неявная реализация интерфейсов
- **Java:** `interface`, явная реализация, default methods

#### 5. Указатели
- **Go:** `*T`, `&`, передача по ссылке
- **Java:** ссылки на объекты, примитивные типы

#### 6. Массивы и слайсы
- **Go:** массивы фиксированной длины, слайсы, `append`, `make`
- **Java:** массивы, `ArrayList`, `List`

#### 7. Карты (Maps)
- **Go:** `map[K]V`, проверка существования ключа
- **Java:** `HashMap`, `Map`, `getOrDefault`

#### 8. Ошибки
- **Go:** `error`, множественные возвращаемые значения
- **Java:** исключения, `try-catch`, `Optional`

#### 9. Конкурентность
- **Go:** горутины, каналы, `select`
- **Java:** `Thread`, `ExecutorService`, `CompletableFuture`

#### 10. HTTP-сервер
- **Go:** `net/http`, маршрутизация, обработчики
- **Java:** Spring Boot, `@RestController`, JAX-RS

#### 11. JSON
- **Go:** `encoding/json`, теги struct
- **Java:** Jackson, Gson, JSON-B

#### 12. Работа с файлами
- **Go:** `os`, `io/ioutil`
- **Java:** `java.nio.file`, Files API

#### 13. CLI-приложения
- **Go:** флаги, `os.Args`, Cobra
- **Java:** `args4j`, Picocli, Spring Shell

#### 14. Мокирование
- **Go:** интерфейсы для тестирования, testify
- **Java:** Mockito, EasyMock

#### 15. Бенчмарки
- **Go:** `testing.B`, Benchmark функции
- **Java:** JMH

#### 16. Рефлексия
- **Go:** пакет `reflect`
- **Java:** `java.lang.reflect`

#### 17. Контекст (Context)
- **Go:** `context.Context`, таймауты, отмена
- **Java:** `ExecutorService` shutdown, Project Reactor

#### 18. Веб-сервер с базой данных
- **Go:** `database/sql`, SQLite/PostgreSQL
- **Java:** Spring Data JPA, JDBC

#### 19. Deploy
- **Go:** Docker, бинарники, cross-compilation
- **Java:** Docker, JAR, GraalVM native image

## Формат

- Markdown (.md) для каждой главы
- Код на Go и Java с комментариями
- Примеры тестов на обоих языках
- Практические упражнения
